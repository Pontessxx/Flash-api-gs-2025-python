# Como funciona o projeto

API em FLASH API, para rodar basta abrir o cmd na pasta raiz e escrever:
```cmd
    pip install -r requirements.txt
```
Para rodar a api:
```cmd
    uvicorn main:app --host 0.0.0.0 --port 8000  --reload
```

## Para rodar o app
```cmd
    cd /MeuAppTS/
```
```cmd
    npm install
```
```cmd
    npx expo start
```
